# Guide — Employee Attrition Risk Analysis

## Prerequisites

- Python 3.10+ (for Python projects) / Node.js 18+ (for Next.js projects)
- Git

## Setup

```bash
# Clone the repository
git clone https://github.com/keerthishree20/Employee-attrition-risk-analysis.git
cd Employee-attrition-risk-analysis
```

## Running the Notebooks

```bash
pip install -r requirements.txt
jupyter notebook
```
Open the `.ipynb` files in the browser.
